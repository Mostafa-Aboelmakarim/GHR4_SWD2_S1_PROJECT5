# Shop Filtering Web Application

This project is a mini team project that builds a simple online shop interface.  
The application allows users to browse products and filter them by name and price using API data.

## Features
- Display products in a card layout
- Filter products by name
- Filter products by minimum and maximum price
- Dynamic product loading using API

## Technologies Used
- HTML
- CSS
- JavaScript
- API integration

## Project Goal
The goal of this project is to practice frontend development and teamwork while building a small e-commerce style interface.
